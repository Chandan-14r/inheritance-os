# Inheritance OS

A modern web application to plan, protect, and pass down your financial legacy to future generations.
Winner template setup for hackathons.

## Architecture
- **Frontend**: React + Vite + TailwindCSS + Recharts
- **Backend**: Node.js + Express + MongoDB
- **AI Integration**: OpenAI (GPT-4o-mini) for personalized letter generation & tax advice.

## Quick Start

### Backend
1. `cd backend`
2. `npm install`
3. Update `.env` with a real OpenAI API key if you want letter generation to work.
4. Make sure MongoDB is running locally on port 27017.
5. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
4. Visit `http://localhost:5173`

Enjoy building your MVP!
